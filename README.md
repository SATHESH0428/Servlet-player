# Servlet Player Management System

This project is a **Player Management System** developed using **Java Servlets** and **JSP**.  
It performs basic **CRUD operations** (Create, Read, Update, Delete) on player data using a database.

This project demonstrates core **Servlet, JSP, and JDBC concepts**.
